Agent eksplorasi codebase harus struktural, bukan sekadar search random

Agent eksplorasi codebase yang efektif harus memahami struktur sistem, file penting, relasi modul, dan alur domain, bukan sekadar mencari potongan kode secara acak.

Catatan gue:

  • Eksplorasi terstruktur memberi agent konteks sistemik yang lebih besar.
  • Search random sering menghasilkan solusi lokal yang sintaksnya benar tetapi arsitekturalnya salah.
  • Agent perlu diberi instruksi eksplisit untuk melakukan eksplorasi sebelum mengubah kode.

Implikasi:

  • Instruksi agent sebaiknya memuat tujuan eksplorasi dan area penting yang harus dieksplorasi.
  • Eksplorasi terstruktur juga membantu agent menghasilkan rekomendasi yang lebih konsisten.
  • Ini menambah lapisan governance di atas kemampuan AI, bukan hanya menambah konteks repo.
